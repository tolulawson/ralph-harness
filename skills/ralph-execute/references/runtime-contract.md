# Ralph Runtime Contract

Source of truth for an installed harness:

1. `AGENTS.md` as the Codex loader
2. `.ralph/constitution.md` as the durable doctrine
3. `.ralph/policy/project-policy.md` as the repo-specific workflow policy
4. `.ralph/state/workflow-state.json` as the canonical runtime state
5. `.ralph/state/spec-queue.json` as the canonical spec queue
6. `specs/INDEX.md` as the human-readable spec register
7. `.ralph/reports/` as the role handoff history
8. `.ralph/logs/events.jsonl` as the append-only audit trail
